Getting Started

Welcome to Masjid Tools! This guide will help you set up your prayer times widget in just a few minutes.

Complete Setup Guide

Step 1: Download the Template

First, download our prayer times template file:

üì• Download CSV Template

This file contains sample prayer times that you'll replace with your mosque's times.

Step 2: Create Your Google Sheet

  1. Go to sheets.google.com
  2. Click "Blank" to create a new spreadsheet
  3. Click "File" ‚Üí "Import"
  4. Click "Upload" and select the CSV file you downloaded
  5. Choose "Replace spreadsheet" and click "Import data"

You now have a Google Sheet with the correct format for prayer times.

Step 3: Add Your Prayer Times

  1. Replace the sample data with your mosque's prayer times
  2. Update the Month column (Column A) with actual months
  3. Update the Day column (Column B) with actual days
  4. Fill in your prayer times in columns C-G (Fajr, Dhuhr, Asr, Maghrib, Isha)
  5. Add Jumuah times in columns J-K if you have multiple Friday prayers

You can use either 12-hour format (1:30 PM) or 24-hour format (13:30).

Step 4: Make Your Sheet Public

  1. Click the "Share" button (top-right corner)
  2. Click "Change to anyone with the link"
  3. Set permissions to "Viewer"
  4. Click "Done"
  5. Copy the Google Sheet URL (it should look like: https://docs.google.com/spreadsheets/d/YOUR_ID/edit)

This allows the widget to read your prayer times data.

Step 5: Customize Your Widget

  1. Use our interactive demo to choose your colors and settings
  2. Paste your Google Sheet URL into the "Google Sheet URL" field
  3. Enter your mosque name and location
  4. Choose whether to show Athan or Iqama times
  5. Select how many Jumuah prayers you have
  6. Click "Update Widget" to see your changes

The preview will update instantly as you make changes.

Step 6: Copy Your Widget Code

  1. Click the "Copy Code" button in the demo
  2. The generated code will appear in the box
  3. Click "Copy to Clipboard" to copy the code

This code contains all your settings and will work on any website.

Step 7: Add to Your Website

WordPress Websites:

  1. Go to your WordPress admin dashboard
  2. Navigate to "Appearance" ‚Üí "Widgets" or "Customize"
  3. Add a "Custom HTML" widget
  4. Paste your copied code into the widget
  5. Save and publish your changes

HTML Websites:

  1. Open your website's HTML file in a text editor
  2. Find where you want the widget to appear
  3. Paste your copied code at that location
  4. Save the file and upload it to your web server

Website Builders (Wix, Squarespace, etc.):

  1. Add an "Embed" or "HTML" element to your page
  2. Paste your copied code into the embed box
  3. Save and publish your page

The widget will automatically update daily with your latest prayer times!

location string '' City and country for display

accentColor string '#ffffff' Text and accent color
timeType string 'athan' 'athan' or 'iqama' - which times to display

Google Sheets Setup

Setting up your Google Sheet for prayer times data:

Sheet Structure

Your Google Sheet should have the following structure:

Main Prayer Times (Columns A-H)

  • Column A: Date (MM/DD/YYYY format)
  • Column B: Fajr time
  • Column C: Dhuhr time
  • Column D: Asr time
  • Column E: Maghrib time
  • Column F: Isha time
  • Column G: Additional notes (optional)
  • Column H: Notes (optional)

Jumuah Times (Columns J-K)

  • Column J: First Jumuah time
  • Column K: Second Jumuah time (if applicable)
  • Column L: Third Jumuah time (if applicable)

Making Your Sheet Public

  1. Open your Google Sheet
  2. Click "Share" in the top-right corner
  3. Change permissions to "Anyone with the link can view"
  4. Copy the sharing URL
Important: Make sure your sheet is set to "Anyone with the link can view" for the widget to work properly.

Troubleshooting

Common issues and solutions:

Widget not loading

Possible causes:

  • Google Sheet is not public
  • Invalid Google Sheet URL
  • Network connectivity issues

Solutions:

  • Check that your Google Sheet is set to "Anyone with the link can view"
  • Verify the Google Sheet URL is correct
  • Check browser console for error messages

Times showing as "--:--"

Possible causes:

  • No data for current date
  • Incorrect date format in Google Sheet
  • Empty cells in the sheet

Solutions:

  • Add data for the current date
  • Use MM/DD/YYYY format for dates
  • Ensure all time cells have values

Jumuah times not showing

Possible causes:

  • Jumuah times not in correct columns (J, K, L)
  • jumuahCount setting doesn't match available data

Solutions:

  • Place Jumuah times in columns J, K, L
  • Set jumuahCount to match your data
