Getting Started

Welcome to Masjid Tools! This guide will help you set up your prayer times widget in just a few minutes.

Quick Start

  1. Download the template: Download CSV Template
  2. Set up Google Sheets: Import the template and make it public
  3. Customize your widget: Use our interactive demo
  4. Embed on your website: Copy the generated code

Basic Embed Code

<!-- Prayer Times Widget -->
<div id="iqama-widget-container"></div>

<script>
window.IqamaWidgetConfig = {
    googleSheetUrl: 'YOUR_GOOGLE_SHEET_URL',
    title: 'Your Masjid Name',
    location: 'Your City, Country',
    backgroundColor: '#1a1a1a',
    accentColor: '#ffffff',
    timeType: 'athan',
    jumuahCount: 1
};

const script = document.createElement('script');
script.src = 'https://ilyasarif100.github.io/Masjid-Tools/dist/prayer-times-widget.js';
document.head.appendChild(script);
</script>

Configuration Options

All configuration options for the prayer times widget:

Option Type Default Description
googleSheetUrl string required URL of your Google Sheet with prayer times
title string '' Masjid or organization name
location string '' City and country for display
backgroundColor string '#1a1a1a' Background color of the widget
accentColor string '#ffffff' Text and accent color
timeType string 'athan' 'athan' or 'iqama' - which times to display
jumuahCount number 1 Number of Jumuah prayer times (1-3)

Google Sheets Setup

Setting up your Google Sheet for prayer times data:

Sheet Structure

Your Google Sheet should have the following structure:

Main Prayer Times (Columns A-H)

  • Column A: Date (MM/DD/YYYY format)
  • Column B: Fajr time
  • Column C: Dhuhr time
  • Column D: Asr time
  • Column E: Maghrib time
  • Column F: Isha time
  • Column G: Additional notes (optional)
  • Column H: Notes (optional)

Jumuah Times (Columns J-K)

  • Column J: First Jumuah time
  • Column K: Second Jumuah time (if applicable)
  • Column L: Third Jumuah time (if applicable)

Making Your Sheet Public

  1. Open your Google Sheet
  2. Click "Share" in the top-right corner
  3. Change permissions to "Anyone with the link can view"
  4. Copy the sharing URL
Important: Make sure your sheet is set to "Anyone with the link can view" for the widget to work properly.

Embedding the Widget

How to embed the prayer times widget on your website:

Method 1: Direct Embed

Copy the generated code from our demo page and paste it into your website's HTML.

Method 2: WordPress

  1. Go to your WordPress admin panel
  2. Navigate to Pages or Posts
  3. Switch to "Text" or "HTML" editor
  4. Paste the embed code
  5. Save and publish

Method 3: HTML Websites

Add the embed code to your HTML file where you want the widget to appear:

<!-- Add this where you want the widget -->
<div id="iqama-widget-container"></div>

<!-- Add this before closing </body> tag -->
<script>

    // Your configuration here
};
const script = document.createElement('script');
script.src = 'https://ilyasarif100.github.io/Masjid-Tools/dist/prayer-times-widget.js';
document.head.appendChild(script);
</script>

Troubleshooting

Common issues and solutions:

Widget not loading

Possible causes:

  • Google Sheet is not public
  • Invalid Google Sheet URL
  • Network connectivity issues

Solutions:

  • Check that your Google Sheet is set to "Anyone with the link can view"
  • Verify the Google Sheet URL is correct
  • Check browser console for error messages

Times showing as "--:--"

Possible causes:

  • No data for current date
  • Incorrect date format in Google Sheet
  • Empty cells in the sheet

Solutions:

  • Add data for the current date
  • Use MM/DD/YYYY format for dates
  • Ensure all time cells have values

Jumuah times not showing

Possible causes:

  • Jumuah times not in correct columns (J, K, L)
  • jumuahCount setting doesn't match available data

Solutions:

  • Place Jumuah times in columns J, K, L
  • Set jumuahCount to match your data

API Reference

Advanced configuration and API details:

Global Functions

createWidget()

Manually create or update the widget with current configuration.

window.createWidget();

Events

widgetLoaded

Fired when the widget has finished loading.

document.addEventListener('widgetLoaded', function(event) {
    console.log('Widget loaded successfully');
});

Data Format

The widget expects prayer times in 12-hour format (e.g., "5:30 AM", "7:45 PM").

// Supported time formats:
"5:30 AM"     // ✅ Correct
"17:30"       // ❌ 24-hour format not supported
"5:30"        // ❌ Missing AM/PM
"05:30 AM"    // ✅ Correct with leading zero
